Projet

Apprendre avec la Science - automatisation des revisions

Categorie : Projets Data Statut : termine

Objectif : transformer une recherche personnelle sur la memorisation efficace en un outil capable de planifier les revisions, recuperer les contenus du jour, et generer automatiquement un auto-test pertinent diffuse sur Discord.

Voir le GitHub
Python Excel Discord Playwright Beautiful Soup LLM API Webhook Make / Integromat

Genese

L'apprentissage est un sujet central pour moi. J'ai donc decide de me former serieusement aux meilleures techniques de memorisation et d'etudier la litterature scientifique sur le sujet. Cette recherche m'a pousse a comprendre concretement ce qui se passe lorsqu'on apprend : selection de l'information, memoire de travail, stockage en memoire long terme, recuperation, consolidation et courbe d'Ebbinghaus.

En pratique, cela m'a amene a comparer les methodes d'apprentissage les plus efficaces, a distinguer les faux bons conseils des approches vraiment solides, puis a concevoir un outil capable d'automatiser ce travail de revision. Le premier resultat a pris la forme d'un prototype no-code construit avec Make / Integromat, que j'ai presente dans la video ci-dessous.

Cette premiere version expose deja le coeur de la demarche : s'appuyer sur les meilleures pratiques issues de la recherche, utiliser la repetition espacee au bon moment, privilegier l'auto-test plutot que la simple relecture, et supprimer au maximum la friction operationnelle pour que reviser devienne quelque chose de faisable au quotidien.

Adaptation pour la Wild Code School

Plus tard, pendant mon cursus pour apprendre le metier de Data Analyst a la Wild Code School, j'ai presente cet outil aux responsables de l'ecole et je leur ai propose une adaptation sur mesure pour leurs besoins. Ils m'ont fait confiance et m'ont laisse carte blanche pour le mettre en place, y compris l'acces a leur serveur Discord.

Pipeline automatise

J'ai donc repris ce principe en construisant un tableau Excel centralisant tous les cours, avec calcul automatise des dates de revision optimales a partir de la courbe d'Ebbinghaus.

Excel des cours → Calcul Ebbinghaus → Scraping → LLM → Webhook Discord

Chaine de traitement

  • L'outil repere les cours a reviser du jour a partir du planning calcule.
  • Il recupere les URLs des cours concernes, puis scrape le contenu des pages, documents et sources externes avec Playwright et Beautiful Soup.
  • Le contenu agrege part ensuite dans un modele LLM configure sur mesure pour produire un auto-test d'environ 20 minutes couvrant l'ensemble des notions a reviser.
  • Le resultat final est envoye par webhook dans un salon Discord dedie aux promos Data Analyst de septembre 2025.

Competences demontrees

Ce projet combine recherche appliquee en sciences cognitives, structuration de contenu pedagogique, automatisation de processus, scraping web, orchestration d'API, configuration LLM, generation d'auto-tests et mise en production d'un usage concret dans un environnement collectif.
